# 🎬 Movie Review App

> A modern movie review web application where users can securely log in, add reviews, and manage their favorite movie ratings with a clean and interactive interface.

---

## ✨ About The Project

The **Movie Review App** is a full-stack web application built using **HTML, CSS, JavaScript, Node.js, Express, MongoDB, and JWT Authentication**.

Users can:
- Create reviews for movies 🎥
- Select movie genres 🍿
- Add ratings ⭐
- Write comments ✍
- Securely log in using authentication 🔐

The application stores reviews locally and supports backend user authentication using MongoDB and JWT.

## ▶ Run Locally

### Clone Repository

```bash
git clone https://github.com/your-username/movie-review-app.git
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
npm start
```

### Run Application

Open in browser:

```text
http://localhost:5000
```
